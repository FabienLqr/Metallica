# Metallica

Why?

This project is designed to enhance my frontend development skills, focusing on a topic that I am passionate about.
I'm a Metallica fan since a few years and give honor to this amazing band motivated me to do this.
In this website you can view all Metallica's Album and all songs of each album.

How?

Je developpe ce project avec React.js et SASS pour le style. D'autres package sont prevu, dont un pour pouvoir lancer la musique sur laquelle l'utilisateur se situe afin d'avoir une immersion musicale lors de la visite du site. Ceci est possible grace au package react-h5-audio-player.

Lorsque l'on est sur la page d'un album, nous persevons des boutons nous permettant d'avoir les lyrics ainsi que la musique qui se lance. Ceci sont d'un jaune petant et nous pouvons nous demander pourquoi? La reponse est relativement simple: c'est la couleur principale du dernier album de Metallica, 72 seasons. Lorsque le curseur passe sur le bouton celui devient plus pale, comme le centre de l'album 72 seasons. La couleur de la police, noire, rapelle encore une fois la couleur de l'album. Ceci contraste avec la couleur rouge pour les couplets et refrains qui vient eux du premier album studio du groupe Kill'em all. 
Ceci rappelle d'ou a commence le groupe et la ou il en est actuellement.
